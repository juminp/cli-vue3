# cli-vue3

## Project create

```
vue create <project-name>
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
